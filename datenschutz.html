2. Datenerhebung und -speicherung

Fishfy erhebt und speichert folgende Daten:

  • Standortdaten: Nur bei aktiver Nutzung der App, um Wetterdaten für den aktuellen Standort abzurufen und Angelspots zu markieren
  • Fotos: Fotos von Fängen werden ausschließlich lokal auf dem Gerät gespeichert
  • Angelspots und Notizen: Alle Spots und Notizen werden lokal auf dem Gerät gespeichert
  • Wetterdaten: Werden über Apple WeatherKit API abgerufen und lokal für maximal 72 Stunden gespeichert
  • Fangdaten mit Wetterbedingungen: Werden lokal auf dem Gerät gespeichert

3. Datenübertragung

Alle Daten werden ausschließlich lokal auf Ihrem Gerät gespeichert. Es erfolgt keine Übertragung Ihrer persönlichen Daten an externe Server, außer:

  • Apple WeatherKit: Wetterdaten werden über Apple WeatherKit API abgerufen. Diese Datenübertragung erfolgt gemäß den Datenschutzbestimmungen von Apple.
  • Freiwillige Datenweitergabe: Optional können Sie zustimmen, dass anonymisierte Fangdaten automatisch an uns übermittelt werden, um die App zu verbessern. Diese Funktion kann jederzeit in den Einstellungen deaktiviert werden. Die Daten enthalten keine personenbezogenen Informationen.

4. Standortdaten

Die App verwendet Standortdaten nur während der aktiven Nutzung, um:

  • Wetterdaten für Ihren aktuellen Standort abzurufen
  • Angelspots zu markieren
  • Fangpositionen zu dokumentieren

Standortdaten werden nicht dauerhaft gespeichert. Sie können die Standortfreigabe jederzeit in den iOS-Einstellungen deaktivieren.

5. Kamera

Die App benötigt Zugriff auf die Kamera, um Fänge zu fotografieren. Fotos werden ausschließlich lokal auf Ihrem Gerät gespeichert und nicht an Dritte übertragen.

6. Wetterdaten

Die App nutzt Apple WeatherKit, um Wetterdaten abzurufen. Diese Daten werden von Apple bereitgestellt und unterliegen den Datenschutzbestimmungen von Apple.

Wetterdaten werden lokal für maximal 72 Stunden gespeichert, um Wetterverläufe zu analysieren.

7. Datenexport

Sie können Ihre Daten jederzeit als PDF exportieren. Der Export erfolgt lokal auf Ihrem Gerät.

8. Freiwillige Datenweitergabe

Sie können optional zustimmen, dass Ihre anonymisierten Fangdaten automatisch an uns übermittelt werden, um die App zu verbessern. Diese Funktion kann jederzeit in den Einstellungen deaktiviert werden.

Die Daten werden nur mit Ihrer ausdrücklichen Zustimmung versendet und dienen ausschließlich der Analyse und Verbesserung der App. Sie können diese Einwilligung jederzeit widerrufen.

Die übermittelten Daten enthalten keine personenbezogenen Informationen und werden anonymisiert verarbeitet.

9. Datenlöschung

Sie können alle Daten jederzeit über die App-Funktion "Alle Daten löschen" entfernen. Bei Deinstallation der App werden alle lokal gespeicherten Daten gelöscht.
